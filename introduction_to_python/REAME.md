# Introduction to Python Programming

## Overview

This repository contains my notes, exercises, and projects completed while taking the **Introduction to Python Programming** course on Udacity.

The course provides a solid foundation in Python and focuses on developing practical programming skills through hands-on exercises and real-world examples.

## Course Objectives

Throughout this course, I learned how to:

* Understand Python syntax and programming fundamentals
* Work with variables and data types
* Use operators and expressions
* Control program flow with conditional statements
* Create loops using `for` and `while`
* Define and use functions
* Work with strings and collections such as lists, tuples, dictionaries, and sets
* Read from and write to files
* Handle exceptions and errors
* Import and use Python modules
* Write clean, readable, and reusable code
